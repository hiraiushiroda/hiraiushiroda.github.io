# 30-Day Learning Plan (React + JavaScript + 将来のBackend)
対象: このリポジトリ（Vite + React）
前提: Gitの基本操作はできる
学習時間: 毎日60〜90分 とりあえず今は30日間プランだけを表示している　この30日プランを全クリアしてから、自分の希望を叶える

---

## 0. まず最初に理解すること

あなたの今のリポジトリは「Reactで1ページを表示する最小構成」です。  
最初の目標は、以下を自力で説明できることです。

1. どこから画面描画が始まるか（`src/main.jsx`）
2. 実際のページ本体はどこか（`src/App.jsx`）
3. 見た目の定義はどこか（`src/index.css`, `src/App.css`）
4. 開発サーバーは何で起動するか（`npm run dev`）
5. 公開用ビルドは何で作るか（`npm run build`）

---

## 1. 毎日の進め方テンプレ（固定）

毎日この順でやる:

1. 5分: 前日の `dev-log.md` を読む
2. 5分: 今日の目的を1行で書く
3. 40〜60分: 実装・修正
4. 10分: 動作確認（ブラウザ + コンソールエラー確認）
5. 10分: `dev-log.md` 記録
6. 5分: 明日の最初の1手を書く

## 2. Day 1〜30 超具体プラン

### Day 1: 現状把握
- やること:
  - `src/main.jsx` を読む
  - `src/App.jsx` を読む
  - `npm run dev` で起動
- ゴール:
  - 「ReactDOMがAppを描画している」と説明できる

### Day 2: JavaScript超基礎1
- やること:
  - `const`, `let` の使い分け
  - 配列 `[]` とオブジェクト `{}` の練習
- ミニ課題:
  - 自己紹介データをJSオブジェクトで書く
- ゴール:
  - `name`, `age`, `from` を1つのオブジェクトで扱える

### Day 3: JavaScript超基礎2
- やること:
  - 関数（通常関数/アロー関数）
  - `if`, `for`, `map`
- ミニ課題:
  - 趣味リスト配列を `map` で表示する準備
- ゴール:
  - `map` で表示用データを作る感覚を持つ

### Day 4: React基礎1（JSX）
- やること:
  - JSXの書き方（`className`, `{}`）
  - `App.jsx` の静的文を少し書き換える
- ゴール:
  - JSXは「HTMLっぽいJavaScript」と理解する

### Day 5: React基礎2（コンポーネント分割）
- やること:
  - `Profile` コンポーネントを作る（自分で）
  - `App` から呼ぶ
- ゴール:
  - 「1ファイルに全部書かない」を体感する

### Day 6: React基礎3（props）
- やること:
  - `Profile` に `name`, `age` を props で渡す
- ゴール:
  - 親→子へデータ受け渡しができる

### Day 7: 週次復習1
- やること:
  - ここまでを整理して `README` かメモに要約
  - `npm run lint` 実行
- ゴール:
  - 1週間分を口頭説明できる
